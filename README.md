# ZeroBin – Smart Food Donation Platform

ZeroBin is a full-stack MERN web application designed to reduce food waste by connecting food donors, NGOs, and administrators through a centralized digital platform. The system enables donors to list surplus food, NGOs to request donations, and administrators to manage and monitor the entire donation lifecycle.

The platform simplifies food redistribution, minimizes manual coordination, and promotes sustainable food sharing within communities.

## Features

- User Registration & Secure Login
- Role-Based Authentication (Donor, NGO, Admin)
- Food Donation Listing
- NGO Food Request Management
- Donation Approval Workflow
- Admin Dashboard
- Real-Time Donation Status Tracking
- Responsive User Interface
- REST API Integration
